World War One Roll of Honour

When the World War 1 menu item is clicked, this page is displayed, and a sub menu appears. This page is the Roll of Honour page for the World War One fallen.

The sub menu has three items:

  • Roll of Honour lists the fallen of World War One.

  • Map shows where the fallen of World War One lived.

  • 2018 Exhibition is a page that allows three PDFs from the 2018 library exhibition to be downloaded and viewed.

The menu was done this way because it was easy for me to implement. It can, and probably will change. This is only an ideas website.

Whenever one of these pages is displayed, the header title changes to Ulverston Fallen: World War One.

The first change to this page will be to display a list of the fallen, as it does now. Clicking one of the fallen results in a pop up appearing, giving details of the person selected (just like the current map page does). It will no longer go to another page (except when Java Script is disabled). This is the easy bit, the very easy and quick to do bit.

The second change is much, much more complicated, and that is to add the filtering. Filtering will work on the fallen listed on this page, changing the names in the list.

Current filtering is specific to your website. I want to make it generic so any one can use it. I also want to blog about what I am trying to achieve with lists and filtering. This means writing some very interesting software, in effect, a code generator. This is code I want to be open source.
