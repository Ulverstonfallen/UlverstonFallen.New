Ulverston Fallen Information

This page of the website contains our terms and conditions, cookies and privacy statements, along with other useful information.

Privacy Statement – for Visitors to the Website

This website is a non-profit making personal web site. It aims to be a permanent record of the Ulverston people killed in action during World Wars One and Two.

The only personal information stored is your name, email address, phone number and if required address; information supplied by you when contacting me, using the site’s contact page and in subsequent emails and phone calls. It will only be used for contact purposes and will never be passed to third parties or used for marketing purposes.

Privacy Statement – the Fallen

Information on the fallen comes from public sources, relatives and others who have known the fallen.

I have assumed that all information on this site is correct. If you think the information on a given fallen person is incorrect, please contact me so I can set the record straight.

If you think some of the information about a fallen person should not be on the site, or you are a relative and do not wish the information to be on it, please contact me.

I have tried to ensure that all photographs used on this site are not infringing copyright. Some pictures are from my own personal archive or those of friends; some come from the public domain; and some come from relatives and friends of a fallen person. If you think a picture does infringe copyright, please contact me me so it can be investigated.

If you have any information on a fallen person, please contact me.

Any supplied information will only be used by this web site. If we think that information and photos would be useful for other sites; we will contact, you.

Copyright

Information on this website comes from many sources, hopefully we have permission to use the copyrighted information, especially the pictures.

I do not mind people using the information on this website for educational purposes or their own personal use. Businesses, charities and profit-making organisations need permission to use the content of this web site.

Some information has been supplied by individuals and is not in the public domain. If the individual supplying the information does not want it to be used anywhere else, we will indicate this.

Images on the web site must not be used without our permission. Some pictures on this site may have been supplied from a personal archive for our use only.

Cookies

The website does not use cookies.

The website does not use local storage to store any personal data about you, such as your email address, name, or phone number.

The filtering function uses local storage of your browser to remember your filter criteria. If you do not clear your browser’s history and cache, your filter criteria will be remembered when you return to the browser. The local storage is only available on this browser. If you change browser on your current computer, or you move to one on another computer, your filter criteria will be lost.

Java Script

This website uses Java Script (a programming language) to improve functionality. You might not have heard of it, but don’t worry, most people haven’t.

Virtually all modern websites use Java Script in one way or another. Some are impossible to use if it is disabled. Most will still be usable, but with reduced functionality. The approach of this website is to allow you to view details of the fallen. However, you will not be able to filter the various lists of fallen or use the map pages, as these are heavily dependent on Java Script.

You can determine if Java Script is available by going to the World War 1 Map page, it will tell you:

  • That Java script is not available, because it has been turned off for this website. This is probably because you are aware of Java Script and have chosen to disable it. To get full functionality, you need to enable Java Script for this website. If you haven’t disabled it or know nothing about it, then someone who uses your computer has almost certainly disabled it. To get full functionality, you need to get that person to enable it for this, or even all websites.

  • That you are using an old browser such as Microsoft’s Internet Explorer. In this case the browser is using an old version of Java Script that is unable to run our Java Script.

  • That Java Script is available, and you have full functionality.
