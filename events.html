Upcoming Events

One thing I have always wanted to do is to allow people to create an events page on their website. I want the updating of the page to be automatic, running every night. Removing events that have passed (there is nothing worse than an events page that has events in the past). And adding new ones at specific times (allowing them to be created in advance). If there are no events, the page says, there are no events.

I am pretty sure I can do this with Git Hub pages. As far as I can see, I can run a process every night on the GitHub server (not on your home computer) to update the website.

You don’t have to have an events page, it’s up to you.
